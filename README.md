# Cactus SaaS Automation Bot Suite 🌵

Professional multi-threaded broadcasting and AI assistant system for Cactus (كاكتس) brand.

## Features
- Full User Registration & Admin Approval Gate.
- Multi-threaded interaction handling to prevent button freezing.
- Persistent local database using auto-generated JSON structure.

## Deployment
To launch the live polling engine, run:
`python bot.py`
